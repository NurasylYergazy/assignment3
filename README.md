This project implements a robust system for representing and traversing directed graphs using an Adjacency List structure.
The system is designed to demonstrate the fundamental differences between Breadth-First Search (BFS) and Depth-First Search (DFS) 
algorithms through empirical performance testing.


Classes
- Vertex: Stores the node ID.
- Edge: Connects a source vertex to a destination vertex.
- Graph: Contains the adjacency list and the BFS/DFS methods.
- Experiment: Generates graphs of size 10, 30, and 100 and measures execution time.
- Main: The main entry point to run the code.


 ###Algorithms

### BFS
BFS uses a queue to visit nodes level by level. It is useful for finding the shortest path in unweighted graphs. The complexity is O(V + E).

### DFS
DFS uses recursion to visit nodes by going as deep as possible into a branch. It is useful for tasks like detecting cycles. The complexity is O(V + E).
